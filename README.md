## 1. ABOUT PROJECT

**0. About Project**: Project is focusing on support stakeholders to monitor operational indicators with the goal of boosting revenue, identifying abnormalities, and discovering patterns.

**1. About Dataset:**
This dataset is the sales data of a retail business with full orders as usual.

**2. Quick View:**
- **PDF Format:** You can check it out Dashboard in PDF format at:

    - Path: `./UI-UX-Sales-Monitor-Dashboard-PDF.pdf`
    - Link: `https://github.com/jane-with-data/UI-UX-Sales-Monitor-Dashboard/blob/main/UI-UX-Sales-Monitor-Dashboard-PDF.pdf`
- **Live Dashboard:** You can view the dashboard directly on PBI Service at:
`https://app.powerbi.com/view?r=eyJrIjoiNDAxYWI0NDEtODM4OC00NWI2LWIwMmEtNTlkZjY5NWQ4M2M4IiwidCI6IjZhYzJhZDA2LTY5MmMtNDY2My1iN2FmLWE5ZmYyYTg2NmQwYyIsImMiOjEwfQ%3D%3D`

**3. Dashboard description**:
Dashboard includes 4 pages:
- **Page 1 - Home:** This page is used for quick navigation also spot top 5 performers.
- **Page 2 - Report:** This page show operational sales metric with advanced filter and various dimensional.
- **Page 3 - Help:** This page contains information for help purpose.
